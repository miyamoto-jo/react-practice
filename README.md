## Try To Move
※ `counter-app` と `hello-world` は簡単なものなので試すなら `todo-app` を是非

1. 実行したいアプリのディレクトリに移動
```
cd todo-app
```
2. `npm install` でアプリにnode moduleをインストールする（初回のみ）
npmコマンドがない場合はbrewなどでインストールしてください。
```
npm install
```
3. `npm start` でアプリを実行する
```
npm start
```
4. `localhost:3000` でアクセスできる
